The official Automatic FFL for WooCommerce plugin

== Description ==
Effortless FFL Shipping

✓ Choose which products require shipment to an FFL

✓ Interactive Google map for customer FFL dealer selection

✓ Enable/disable FFLs

✓ Feature FFLs

✓ Hand pick visible FFLs and/or allow automatic visibility of new type 1 and 2 dealers (big box stores disabled by default)

✓ Support for a mixed cart of firearm products and non-firearm products

✓ FFLs up to date and validated by the ATF

✓ Bulk FFL customizations editing

✓ Support for multiple stores

✓ Add custom information about familiar FFL locations like store hours and rates

✓ Same day Customer Service and Technical Support

== Installation ==
- Install and activate the plugin
- Set your Automatic FFL API and Google licenses under WooCommerce->AutomaticFFL tab.
- Find a product and select the checkbox \'FFL Required\' to ship it for FFL Dealers only
